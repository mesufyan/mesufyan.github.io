Laboratory project

Deep Learning for Particle-Physics Event Classification

Type: Lab projectStack: Python, PyTorch, OptunaData: 7,600 events

A PyTorch neural network that separates signal from background in high-energy physics events — built end to end with custom data pipelines, augmentation, Optuna tuning and proper evaluation.

The task

High-energy physics produces huge numbers of collision events, only some of which are signal. The classic ML problem is to separate signal from background from a set of measured features — a clean setting to build a deep-learning pipeline end to end.

What I did

I implemented a multi-layer neural network in PyTorch for binary classification of high-energy physics events on a 7,600-sample dataset, taking the pipeline from raw data to a fully evaluated model.

  • Wrote a custom Dataset and DataLoader for batching and shuffling.
  • Applied data augmentation to improve generalisation on the limited sample.
  • Tuned architecture and training with Optuna.
  • Evaluated with accuracy, precision, recall and AUC-ROC — not accuracy alone.

Why it matters here

This project is the methodological bridge to my clinical work: the same PyTorch fundamentals, custom data pipelines and rigorous evaluation now underpin the attention-based multimodal models in my radiotherapy QA thesis.
