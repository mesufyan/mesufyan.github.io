Laboratory project

Deep Learning for Particle-Physics Event Classification

Type: Lab projectStack: Python, PyTorch, OptunaData: 7,600 events

A PyTorch neural network that separates signal from background in high-energy physics events — built end to end with custom data pipelines, augmentation, Optuna tuning and proper evaluation.

The task

High-energy physics produces vast numbers of events, only some of which are signal. The classic ML problem is to separate signal from background — a clean setting to build a deep-learning pipeline end to end.

What I did

A multi-layer neural network in PyTorch for binary classification on a 7,600-sample dataset, taken from raw data to a fully evaluated model.

  • Custom Dataset and DataLoader for batching and shuffling.
  • Data augmentation to improve generalisation.
  • Optuna for architecture and training tuning.
  • Evaluation with accuracy, precision, recall and AUC-ROC.
The end-to-end pipeline, from raw events to an evaluated, tuned classifier.

Results

ROC for the classifier — well above the chance diagonal.
